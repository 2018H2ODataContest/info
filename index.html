簡介

水乃維持生命三要素之一，穿天透地的水資料無所不在，要如何發揮水資料的最大價值呢? 透過本次水資料應用競賽，由創意無限、數據分析專長的參賽者們，發揮對水資料的想像，運用各式資料組合，發展出各式水服務，呈現出水資料的價值，使民眾有用水安心無虞的美好生活。

競賽方式


  • 競賽分為兩階段，初賽階段主要為書面審查，需填寫提案構想書，並完成繳交1,000元保證金。
  • 作品必須與競賽主題相關，使用主辦單位提供之公開水資料至少一筆進行創作。
  • 創作可包含但不限於數據收集、數據分析、創新服務等，產生水資料創新服務作品，可為手機應用程式、網頁服務、機器人、應用服務等。
  • 主辦單位將選出20隊進入複賽，並於 10/6 起提供開發環境，詳請將於 10/6 當天之說明會發表。
  • 複賽活動期間將安排技術顧問提供參賽團隊專業諮詢。
  • 進入複賽之隊伍，由10/27 之決選簡報日選出前三名，並於11/10 進行成果發表及頒獎典禮。
  • 參賽隊伍全員至少須出席技術說明會及成果發表會，方可退費。

參考資料

以下是本次競賽活動建議的參考資料與主要使用的開發工具


開放水資料清單
彙整水利署、水務局、台灣自來水公司等單位之公開資料，是提供給本次競賽使用的主要資料來源。


政府資料開放平台
「政府資料開放」(Open Data)為各機關於職權範圍內取得或做成，且依法得公開之各類電子資料，依其需求連結下載及利用。也是本次競賽可使用的參考資料來源。


Microsoft Azure
Microsoft Azure 是微軟的公用雲端服務(Public Cloud Service) 平台，也是提供給本次競賽進入複賽後的參賽者使用的工具。參賽者必須在這個指定的開發環境做出想要的應用。

報名說明


  • 報名資格：對大數據分析有興趣之全國各大專院校，含大學部或研究所在校生及應屆畢業生(畢業一年內)，不限國籍、性別，皆可參與，以團隊為參賽單位(每隊3~5人)。

  • 報名方式 : 由報名隊伍透過報名網頁報名，初賽階段需填寫提案構想書，並於截止日前完成繳費(繳費方式為活動專用金流平台KKTIX)，經主辦單位審核報名資格無誤後，始為報名成功。

  • 報名費用 : 每隊保證金為新台幣1,000元整，報名成員至少須出席技術說明會及成果發表會，方可退費。

  • 備註：主辦單位有最終之資格審查權
